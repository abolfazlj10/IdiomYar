# IdiomYar

IdiomYar is a focused English idiom study app built with Next.js. It helps learners practice idioms with flash cards, read lesson context when needed, create short bilingual stories, and save difficult idioms for review.

## Getting Started

Run the development server:

```bash
npm run dev
```

Then open:

```text
http://localhost:3006
```

## Study Modes

- Flash Cards for quick recall
- Book Study for lesson-by-lesson context
- Story Creator for applying selected idioms in bilingual scenes
- Archive & Review for saved stories, bookmarks, and review cards
